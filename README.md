## 주요 기능

- ☀️ 현재 날씨 정보 ☁️
- 현재 날씨를 기준으로 옷차림 추천
- 현재 날씨를 기준으로 옷차림을 기록
- 현재 날씨를 기준으로 내가 기록했던 옷차림을 확인 할 수 있음.(10개 까지)
- 다른 유저들과 기록 공유 가능
- 온도별로 내 기록을 조회할 수 있는 필터기능
- (프로필)닉네임 수정
